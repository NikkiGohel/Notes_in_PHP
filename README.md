# 📝 Notes in PHP

A simple web-based Notes Management System built using PHP and MySQL.  
This application allows users to create, edit, and delete notes with persistent database storage.

---

## 🚀 Features
- Add new notes
- Edit existing notes
- Delete notes
- Store data in MySQL database
- Simple and clean UI

---

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP (Local Server)

## 🗄 Database Setup
1. Open phpMyAdmin
2. Create new database (example: notes_db)
3. Import `database.sql` file

---

## ▶ How to Run Project
1. Install XAMPP
2. Copy project folder into htdocs
3. Start Apache and MySQL
4. Import database.sql
5. Open browser:
   http://localhost/project-folder-name
